# Oren IA — Fin Frontend

PWA do assistente financeiro Fin da Oren IA.

## Stack
- React 18 + Vite
- PWA (instalável na tela do celular)
- CSS puro (sem dependências de UI)
- Streaming SSE do backend

## Variáveis de ambiente (Vercel)

```
VITE_BACKEND_URL=https://seu-backend.railway.app
```

## Desenvolvimento local

```bash
npm install
npm run dev
```

## Deploy

Conectar repositório no Vercel. Build command: `npm run build`. Output: `dist`.
